## Concept

Le design de mon portfolio est tout droit inspiré de l'univers du jeu vidéo **Silent Hill**, avec deux thèmes :
- **Mode Clair** : On arrive dans l'atmosphère brumeuse signature de la ville...
- **Mode Sombre** : On sort sa lampe de poche et on bascule dans les ténèbres !

### Fonctionnalités actuellement visibles

- **Système de thème** Light/Dark avec transitions fluides
- **Design responsive** optimisé mobile, tablette et desktop
- **Carrousel de projets** interactif avec navigation
- **Effet machine à écrire** sur la description
- **Effets visuels inspirés de Silent Hill** : brouillard, ténèbres, lampe torche
- **Animations CSS** personnalisées
- **Carousel de compétences** défilant automatiquement
- **Section CV** avec version PDF téléchargeable
  
## Technologies utilisées dans le projet

- **HTML5** - Pour la structure sémantique
- **CSS3** - Pour les animations et transitions avancées
- **JavaScript Vanilla** - Pour l'interactivité sans frameworks
- **Font Awesome** - Pour les différentes icônes
- **Google Fonts** - Pour les typographies (Playfair Display, Inter, Indie Flower)
